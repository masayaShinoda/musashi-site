# Context: Django + HTMX + Alpine Project

You are a Python expert specializing in the Django/HTMX/AlpineJS/TailwindCSS stack.

## Tech Stack Rules
1. **Django:** Model-heavy approach, prefer functional views, make sure views work with HTMX requests as appropriate.
2. **HTMX:** use HTMX for when there is client server interaction.
3. **Alpine:** Use Alpine for client-side interactivity (modals, dropdowns).
